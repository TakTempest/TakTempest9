【サイト管理者向け】EmailJS セットアップ手順

このサイトはEmailJSを使い、APIキー不要でブラウザからメールを送信します。


① EmailJSアカウント作成

→ https://www.emailjs.com で無料登録（月200通まで無料）


② Email Serviceを追加

→ ダッシュボード > Email Services > Gmailを連携
→ 表示される Service ID をコピー


③ Email Templateを作成

→ Email Templates > 新規作成
→ To: jgt25yhpp@gmail.com
→ Subject: 【東京陰陽師】運気解釈依頼 - {{client_name}}
→ Body: 下記変数をテンプレートに配置
{{client_name}} {{client_email}} {{q1}} {{q2}} {{q3}} {{q4}} {{q5}} {{style}} {{submitted_at}}
→ 表示される Template ID をコピー


④ Public Keyを取得

→ Account > General > Public Key をコピー


⑤ 下記3行をこのHTMLファイルに設定

→ スクリプト冒頭の以下の定数を書き換えてください：

const EMAILJS_SERVICE_ID = 'your_service_id';
const EMAILJS_TEMPLATE_ID = 'your_template_id';
const EMAILJS_PUBLIC_KEY = 'your_public_key';


⑥ note購読URLを設定

→ const NOTE_PAYMENT_URL = 'https://note.com/あなたのnoteのメンバーシップURL';
